### CSS样式编码格式规范
<br />

**代码风格**

##### 1. 代码格式化

样式书写一般有两种格式：一种是紧凑格式（Compact）
```shell
.className { display: block;width: 50%; }
```
一种是紧凑格式（Expanded）
```shell
.className {
    display: block;
    width: 50%;
}
```
**团队约定：统一使用展开格式书写样式**
