O(nlogn)级别的算法是最优算法时间复杂度，那为什么还要学习O(n^2)的排序算法?
```shell
1. 相对基础

2. 编码简单,易于实现,是一些简单情景的首选

3. 在一些特殊的情况下,简单的排序算法更有效

4. 简单排序算法思想衍生出复杂排序算法

5. 可能成为子过程,改进更复杂的排序算法
```
